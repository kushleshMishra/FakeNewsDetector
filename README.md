# 📰 Fake News Detection Using Machine Learning & NLP

![Python](https://img.shields.io/badge/Python-3.12-blue)
![Machine Learning](https://img.shields.io/badge/Machine%20Learning-Scikit--Learn-orange)
![NLP](https://img.shields.io/badge/NLP-TF--IDF-green)
![Streamlit](https://img.shields.io/badge/Deployment-Streamlit-red)

An end-to-end **Fake News Detection System** built using **Machine Learning, Natural Language Processing (NLP), and Streamlit**.

This project classifies news articles as **Fake or Real** by analyzing textual information using TF-IDF feature extraction and multiple Machine Learning algorithms.

The project covers the complete Machine Learning workflow:

**Data Collection → Data Cleaning → EDA → Text Processing → Model Training → Evaluation → Deployment**
